# Library Management System

A full-stack Library Management System developed using Node.js, Express.js, MySQL, HTML, CSS, and JavaScript. This project helps manage books, users, issuing books, returns, and library records efficiently.

---

## Features

- User Login and Registration
- Add, Update, Delete Books
- Search Books
- Issue and Return Books
- Manage Library Records
- Responsive User Interface
- Database Integration using MySQL

---

## Technologies Used

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MySQL
